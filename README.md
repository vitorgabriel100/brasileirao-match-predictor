# ⚽ Projeto Previsão ML - Brasileirão

Projeto de Machine Learning para prever o resultado de partidas do Campeonato Brasileiro.

## 🚀 Tecnologias
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## 📊 Funcionalidades
- Tratamento dos dados históricos
- Treinamento do modelo
- Previsão de vitória mandante, empate ou vitória visitante
- Interface interativa com Streamlit

## ▶️ Como executar
```bash
python src/limpeza.py
python src/treino.py
streamlit run app.py
